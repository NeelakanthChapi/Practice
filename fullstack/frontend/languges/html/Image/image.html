Image Topic

HTML img tag is used to display image on the web page. HTML img tag is an empty tag
that contains attributes only, closing tags are not used in HTML image element. Images are not
technically inserted into a web page; images are linked to web pages.

The img tag is empty, it contains attributes only, and does not have a closing tag.

The img tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image
