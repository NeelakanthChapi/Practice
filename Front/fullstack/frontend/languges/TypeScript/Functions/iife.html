IIfe

It is a JavaScript function that runs as soon as it defined. An IIFE (Immediately Invoked Function Expression)
can be used for avoiding the variable hoisting from within the blocks. It allows the public access to methods
while retaining the privacy for variables defined in the function.

IIFE is a design pattern that is also known as the Self-Executing Anonymous Function. It contains two major parts:

1.The first part is the anonymous function having a lexical scope, which is enclosed within the Grouping operator ().

2.The second part creates the IIFE by which the JavaScript engine will interpret the function directly.
