Promise

What is promise:-In javascript promise is a good way to handle async operation,
It is used to findout if the async operation is successfuly completed or not.
May have 3 States:-

  • Pending State
  • Fulfilled
  • Rejected

A promise starts in a pending state that means process is not complte.If the process ends in fulfill state.
And if an error occurs,the process ends in Rejected state.
For example,when you request data from the server by using promise,it will be in a pending state.
When DATA arrives successfully it will be in fulfill state.if an error occurs it will be in rejected state.
