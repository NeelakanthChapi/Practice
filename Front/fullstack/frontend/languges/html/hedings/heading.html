What is heading Tag

The heading tag is used in HTML to define headings of a page. Headings are defined by hn , with “n” being
a number between 1 and 6, and determines which position a heading has in the hierarchy of the heading structure.

  • The h1 is an HTML tag that indicates a title on a website

  • It defines the second-level headings on your webpage

  • Defines a level-three heading

  • Represents a level 4 heading in an HTML document

  • Defines the fifth level heading in the HTML document
  • Defines the least important heading
