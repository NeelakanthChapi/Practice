What is Paragraph

Pragraph is nothing but text, this tag is used to define a paragraph in a webpage.

There are two basic types of HTML elements:

  • Block level elements

  • Inline elements

    • Block Level Elements:

    A block-level element always starts on a new line, and the browsers automatically add some space before and after the element.
    A block-level element always takes up the full width available.

    Inline elements:

    Inline elements display in a line. They do not force the text after them to a new line.Some of the inline elemnets are Span tag and Anchor. tag
